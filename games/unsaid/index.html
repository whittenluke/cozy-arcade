How to play

  • Each round has a theme. The hidden word fits that theme.
  • Guess words of the same length. When a letter is correct and in the right position, it appears in the slot. Only exact matches count.
  • Each wrong letter in a guess costs one token. You start with word length Ã— 4 tokens.
  • Reveal every letter to finish. If you run out of tokens first, the game ends.
